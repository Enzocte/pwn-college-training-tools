# pwn.college Training Tools

This repository contains educational scripts and proof-of-concept programs developed while practicing cybersecurity challenges in controlled environments.

## Topics covered

- cryptography
- XOR and one-time pad exercises
- AES ECB oracle attacks
- packet crafting
- ARP spoofing and packet relaying
- basic web security challenge automation

## Repository structure

- `crypto/`: scripts related to XOR, OTP, and AES challenge solving
- `network/`: low-level networking experiments in C and Python
- `web/`: educational scripts for web security exercises
- `utils/`: small helper scripts for challenge interaction

## Context

All scripts were written for educational purposes in local lab or challenge environments in order to better understand protocols, vulnerabilities, and offensive security techniques in a controlled setting.

## Technologies used

- Python
- C
- pwntools
- requests
- scapy
- raw sockets
